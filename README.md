## Abstract
This project introduces an AI-driven system that is used to automatically prepare personalized learning pathways for individuals in India’s vocational education system. The system utilizes learner profile information, including their academic history, current skills, some socio- economic constraints, and the desired job to suggest optimal training programs being conducted based on  industry needs.The system integrates NSQF-aligned qualifications and real-time labor market information to automatically update recommendations based on the latest data. Machine learning algorithms for learner profiling and recommendation foster ongoing system improvements while incorporating user data privacy controls. StringComparison monitoring indicators are used to measure the impact of these advancements. The platform’s design increases its effectiveness and likely adoptability by diverse user groups. Real-time skill demand data help reduce the existing gaps in matching skills to labor market requirements, ultimately increasing utilization and the effectiveness of training. The design allows for enhanced measurement of the overall effectiveness of the career guidance system over time.

## Introduction
India has a great number of different types of vocational training programs available to an individual, but there is no systematic approach in guiding and advising an individual, in a data-driven manner, to help make the right choice. Existing systems take a generic approach and treat every learner as the same in terms of capabilities and expectations. Consequently, learners also face challenges in deciding what type, of course, would benefit them the most. 
This Capstone project aims to build an AI-driven, data-centric, adaptive learning path recommendation engine which identifies comparable instances based on learners’ profile attributes and recommends suitable learning paths, utilizing the data of past students and relations. More equivalently, it draws inferences from the known equivalent companions to recommend the unknown. The approach relies on the use of a recommender system, intelligent software tool, and website. The whole project is called Career Navigator.
## Literature review
Other techniques such as online learning platforms, career guidance systems, and existing recommendation systems of AI also contribute to this approach. Recent researches have cited the implementation of AI and machine learning to develop recommendation systems in online platforms, course contents, and job recommendations using collaborative filtering, content-based filtering, and hybrid recommendations. Frameworks like NSQF also have structured skill levels but its dynamically updated level is not fully utilized in the recommendation system. The proposed system works in a similar approach but integrates AI with standardized skill frameworks and dynamically mapped labor market intelligence.

## Methodology
The system uses machine learning algorithms to analyse learner profiles and generate personalized recommendations. It gathers user information including education, skills, preferences, and career aspirations. This information is analysed through a hybrid recommendation engine that combines content-based and collaborative filtering techniques. The system then matches user profiles to NSQF-aligned courses and incorporates current labour market information to validate recommendations. Post-enrolment, recommendations are monitored and updated based on the progress and performance of users.

## Results and Discussion
The recommendation engine of the system facilitates effective generation of personalized learning paths. The engine designs and updates recommendations based on user profiles including their work experience, education, and existing competencies mapped to skills and expertise categories. The system is capable of generating recommendations focusing on the required, desired, or optional skills users might want to develop further.
Key highlights:
- Enhanced learner experience and engagement
- Improved matching of user’s skills with recommended courses
- Dynamic updates of available courses and skills based on current market demands
Measures used to assess the performance and results include precision, recall, and user feedback.

## Limitation
- Dependence on quality and availability of user data
- Limited access to real-time labor market datasets
- Cold-start problem for new users
- Potential bias in recommendation algorithms
